## Overview
**Gemini Antigravity CLI** is a fast and easy-to-use command-line interface for **Google Gemini** and **Antigravity** AI models.

Perfect for developers who want a powerful AI coding agent directly in the terminal — with support for slash commands, MCP server, plugins, and agentic workflows.

> [!Important]
> Requires a Google API Key or Gemini access.

## Key Features
- Full support for Gemini 2.5 Pro / Flash and Antigravity models
- Advanced AI Agent capabilities
- Slash commands support
- MCP Server & Plugin system
- Fast coding assistance
- Context-aware conversations
- Cross-platform (Windows 11, macOS, Linux)
- Lightweight and fast startup
- Easy migration from Gemini CLI

## Installation
```bash
# Via npm (recommended)
npm install -g gemini-antigravity-cli

# Or download the executable from Releases
```

## Quick Start
```bash
# Run the CLI
antigravity

# Or
gemini
```

Then use commands like:
- `/help` — Show all commands
- `/model` — Switch AI model
- Ask anything directly for coding help
